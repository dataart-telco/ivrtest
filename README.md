# ivrtest server

Simple ivr server for Restcomm ivr perf test

### Usage
```shell
  -h string
        Host (default "10.0.0.102")
  -l string
        Log level: TRACE, INFO (default "INFO")
  -n string
        Test number (default "7777")
  -p int
        Port (default 7090)
  -r string
        Restcomm host (default "127.0.0.1:8080")
  -r-pswd string
        Restcomm password (default "42d8aa7cde9c78c4757862d84620c335")
  -r-user string
        Restcomm user (default "ACae6e420f425248d6a26948c17a9e2acf")
  -res string
        Nginx address (default "127.0.0.1:8080")
  -res-confirm string
        conf wav (default "ivr-confirm.wav")
  -res-msg string
        message wav (default "ivr-message.wav")

```
